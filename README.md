# Segmentação de Países da IMO por Desempenho

Repositório com os materiais do trabalho de **Aprendizado Não Supervisionado**, que analisa o desempenho histórico dos países na Olimpíada Internacional de Matemática (IMO).

## Arquivos

- **`notebook_ans_imo.ipynb`**
  Notebook com a análise completa: pré-processamento, redução de dimensionalidade (PCA e UMAP) e clusterização (K-Means e DBSCAN).

- **`Relatório_ANS.pdf`**
  Relatório escrito do trabalho.

- **`Segmentacao-de-Paises-da-IMO-por-Desempenho.pptx.pdf`**
  Slides utilizados na apresentação.
